# Nathan Ko — Minimal Portfolio

A minimal two‑page portfolio built with Next.js, React, TypeScript, and Tailwind CSS.

## Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```
Open http://localhost:3000 (or the port shown in the terminal).

## Build
```bash
npm run build
npm start
```

## Structure
- Home: `app/page.tsx`
- About: `app/about/page.tsx`
- API (lines of TS): `app/api/lines/route.ts`
- Global styles: `app/globals.css`
